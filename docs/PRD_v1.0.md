## 1. Executive Summary

### Problem Statement

大学生在选择技能培训机构时，面临严重的信任危机：担心课程质量差、师资水平低、价格不透明、就业承诺虚假、机构资质不正规。这导致潜在客户流失，咨询转化率低。

### Proposed Solution

打造一个以"建立信任"为核心的官网首页，通过真实数据展示、学员案例、资质背书、透明的服务承诺等信任元素，降低用户决策门槛,引导用户主动发起咨询。

### Success Criteria

| 指标           | 目标值                   | 衡量方式                    |
| -------------- | ------------------------ | --------------------------- |
| 咨询转化率     | 首页访客 → 咨询提交 ≥ 5% | 埋点统计                    |
| 页面跳出率     | ≤ 40%                    | Google Analytics / 百度统计 |
| 平均停留时长   | ≥ 90 秒                  | 埋点统计                    |
| 咨询表单完成率 | 表单打开 → 提交 ≥ 60%    | 漏斗分析                    |

## 2. User Experience & Functionality

### User Personas

| 角色                         | 特征                             | 核心诉求                           | 信任顾虑                           |
| ---------------------------- | -------------------------------- | ---------------------------------- | ---------------------------------- |
| **大一新生（黄金转化期）**   | 刚入学，学习热情高，时间充裕     | 提前规划、掌握技能、建立竞争优势   | 课程适合零基础吗？学完能达到什么水平？ |
| **大二至大四学生（主力群体）** | 面临实习/校招/升学等现实压力     | 短期内快速提升，拿到满意的 offer 或升学机会 | 培训真的有用吗？时间来得及吗？     |
| **学生家长（决策影响者）**   | 40-55岁，为孩子支付学费          | 投资回报明确、机构可靠、孩子受益   | 会不会是骗子？退费容易吗？         |

### User Stories & Acceptance Criteria

#### Story 1: 快速理解价值主张

> As a 大学生，I want to 在 10 秒内理解 EaglesLab 是做什么的，so that 我能判断是否继续了解。

**Acceptance Criteria:**

- [x] 首屏 Hero 区域清晰展示：「专注大学生升学和就业的解决方案」
- [x] 首屏包含核心数据：10年经验 / 2000+学员 / 85%就业率 / 95%升学率
- [x] 首屏有明确的 CTA 按钮：「免费咨询」或「了解课程」

#### Story 2: 验证机构可信度

> As a 学生家长，I want to 看到机构的资质证明和真实案例，so that 我能放心让孩子报名。

**Acceptance Criteria:**

- [ ] 展示企业资质（营业执照、教育资质等）
- [ ] 展示合作企业 Logo 墙（如有）
- [ ] 展示至少 6 个真实学员案例（含姓名/学校/入职企业/薪资涨幅）
- [ ] 学员评价带头像和真实信息，可点击查看详情

#### Story 3: 了解课程体系

> As a 大学生，I want to 快速了解有哪些课程、适合什么人，so that 我能找到适合自己的方向。

**Acceptance Criteria:**

- [ ] 课程分类清晰：技能培训 / 求职辅导 / 学业辅导
- [ ] 每个课程卡片展示：适合人群、学习周期、核心技能点
- [ ] 提供「不知道学什么？测一测」的互动入口

#### Story 4: 消除价格和退费顾虑

> As a 学生家长，I want to 了解价格区间和退费政策，so that 我知道风险可控。

**Acceptance Criteria:**

- [ ] 展示「7天无理由退款」或类似承诺（如适用）
- [ ] 价格区间透明展示，或引导至咨询获取详细报价
- [ ] 常见问题 FAQ 包含退费相关条款

#### Story 5: 便捷发起咨询

> As a 潜在客户，I want to 用最少的步骤发起咨询，so that 我能快速得到回复。

**Acceptance Criteria:**

- [ ] 页面至少 3 处 CTA 入口（首屏、中部、底部）
- [ ] 咨询表单字段 ≤ 4 个（姓名、电话、意向课程、备注）
- [ ] 支持微信扫码添加顾问（备选方案）
- [ ] 提交后展示「30 分钟内回电」承诺

### Non-Goals（不在本期范围）

- ❌ 用户注册/登录系统
- ❌ 在线支付功能
- ❌ 课程详情页（仅做首页）
- ❌ 学员社区/论坛
- ❌ 在线直播/录播课程播放

## 4. Technical Specifications

### Architecture Overview

- **类型**：静态营销页面（可选 SSR/SSG）
- **推荐技术栈**：
  - 前端：Next.js / Nuxt.js（SEO 友好）或纯 HTML/CSS/JS
  - 样式：Tailwind CSS
  - 表单：集成第三方表单服务（如 Tally、金数据）或自建 API
  - 部署：Vercel / 阿里云 OSS + CDN

### Integration Points

| 集成项                      | 用途           | 优先级 |
| --------------------------- | -------------- | ------ |
| 百度统计 / Google Analytics | 流量与转化分析 | P0     |
| 表单提交 API                | 收集咨询信息   | P0     |
| 企业微信/个人微信           | 扫码添加顾问   | P1     |
| 在线客服（如美洽、智齿）    | 实时咨询       | P2     |

### SEO Requirements

- [ ] 页面 Title：「EaglesLab - 专注大学生升学和就业的解决方案」
- [ ] Meta Description 包含核心关键词
- [ ] 结构化数据标记（Organization、Course）
- [ ] 移动端适配，Core Web Vitals 达标

### Performance Requirements

| 指标                   | 目标   |
| ---------------------- | ------ |
| 首屏加载时间（LCP）    | ≤ 2.5s |
| 可交互时间（TTI）      | ≤ 3.5s |
| Lighthouse Performance | ≥ 90   |

## 5. 信任元素设计策略

### 核心信任公式

```
信任 = 真实数据 + 社会证明 + 风险降低 + 专业形象
```

### 具体实施

| 信任类型     | 实施方式                                       |
| ------------ | ---------------------------------------------- |
| **数据背书** | 10年经验、2000+学员、85%就业率、95%升学率（首屏突出） |
| **社会证明** | 真实学员案例（带照片、学校、入职企业）         |
| **权威背书** | 合作企业 Logo、资质证书、媒体报道（如有）      |
| **风险降低** | 退费政策透明、免费咨询、试听课程（如有）       |
| **专业形象** | 高质量页面设计、清晰的课程体系、专业的内容表述 |

## 6. Risks & Mitigation

| 风险             | 影响         | 缓解措施                                      |
| ---------------- | ------------ | --------------------------------------------- |
| 学员案例数据不足 | 信任元素薄弱 | 优先收集 6-8 个高质量案例，含详细信息         |
| 页面加载慢       | 跳出率高     | 图片压缩、CDN 加速、懒加载                    |
| 咨询表单无人跟进 | 转化浪费     | 对接 CRM 或企业微信，设置自动提醒             |
| 竞品价格战       | 用户流失     | 强调差异化价值（专注大学生、学业+就业双保障） |
